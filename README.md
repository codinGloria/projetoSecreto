<h1 align="center"> Gerenciamento de Funcionários ✨ </h1>

Este é um projeto de gerenciamento de funcionários desenvolvido para um teste prático. O projeto utiliza JavaFX para a interface gráfica e JPA (Java Persistence API) com Hibernate para persistência de dados em um banco de dados MySQL. <br>
Confira o código feito e mais informações logo abaixo :]

## Funcionalidades

- Cadastra novos funcionários de uma lista passada pela empresa:
![image](https://github.com/codinGloria/projetoSecreto/assets/64878674/4661d383-f057-4386-9e78-0e4c7744e233)
- Lista todos os funcionários cadastrados.
- Remove um funcionário específico da lista.
- Aplica um aumento de 10% no salário de todos os funcionários.
- Agrupa os funcionários por função (gerente, operador, etc.) e exibe os nomes dos funcionários de cada grupo.
- Lista os funcionários em ordem alfabética.
- Lista os funcionários que fazem aniversário nos meses de outubro e dezembro.
- Identifica e exibe o funcionário mais velho cadastrado.
- Calcula e exibe o total dos salários de todos os funcionários.
- Calcula e exibe quantos salários mínimos (considerando R$ 1212,00) cada funcionário ganha.
